# Blog

Updates, tutorials, and use cases for the NCR plugin. Comments are powered by [Giscus](https://giscus.app/) (GitHub Discussions). To enable them site-wide, enable [GitHub Discussions](https://docs.github.com/en/repositories/managing-your-repositorys-settings-and-features/enabling-features-for-your-repository#enabling-github-discussions-for-a-repository) for the repo, install the [Giscus app](https://github.com/apps/giscus), then add `repoId` and `categoryId` from [giscus.app](https://giscus.app) to `.vitepress/config.mts` under `themeConfig.giscus`.
