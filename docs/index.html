Why Native Content Relationships?

The Old Way: Post Meta

Standard WP_Query

We hook into `WP_Query` so you can use familiar syntax.

functions.php
$args = [
  'post_type'      => 'event',
  'posts_per_page' => 10,
  // Native Content Relationships Query
  'relationship'   => [
    'id'   => 'event_to_speakers',
    'to'   => $speaker_id, 
  ],
];
$query = new WP_Query( $args );
if ( $query->have_posts() ) {
  while ( $query->have_posts() ) {
    $query->the_post();
    // ... standard loop
  }
}
