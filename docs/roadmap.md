# Roadmap

High-level direction for Native Content Relationships. Priorities may shift based on community feedback and core WordPress evolution.

## Near term

- **Documentation** — Full docs site (this site), API reference, and migration guides.
- **Stability** — Schema and API stability commitment; see the [readme](https://github.com/chetanupare/WP-Native-Content-Relationships#readme).
- **Integrations** — WooCommerce, Elementor, Gutenberg, WPML/Polylang support maintained and documented.

## Future

- **WPGraphQL** — Optional add-on for relationship fields in GraphQL schema.
- **Tutorials** — Real-world use cases and code samples.
- **Ecosystem** — Encourage themes and agencies to adopt NCR as a relationship layer.

## Contributing

Ideas and pull requests are welcome. See [Contributing](/contributing) and the [GitHub repo](https://github.com/chetanupare/WP-Native-Content-Relationships).
