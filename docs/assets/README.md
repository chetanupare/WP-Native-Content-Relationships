# Favicon and SEO Assets

This directory contains favicon files, SEO assets, and integration icons for the GitHub Pages site.

## Files

### Favicon and SEO Assets
- favicon.ico - Main favicon for browsers
- favicon-16x16.png - Small favicon
- favicon-32x32.png - Medium favicon
- apple-touch-icon.png - Apple touch icon
- manifest.json - Web app manifest

### Integration Icons
- wordpress-logo-svgrepo-com.svg - WordPress integration icon
- woocommerce-icon-svgrepo-com.svg - WooCommerce integration icon
- wpml-svgrepo-com.svg - WPML multilingual integration icon
- gutenberg-svgrepo-com.svg - Gutenberg block editor integration icon
- elementor-svgrepo-com.svg - Elementor page builder integration icon
- icon-128x128.png - Main plugin icon from WordPress.org

## Usage

### Favicon and SEO Assets
These files are referenced in the HTML head sections of all pages for:
- Browser bookmarks and tabs
- Mobile app icons
- SEO optimization
- Web app installation

### Integration Icons
These SVG icons are used in the "Works seamlessly with the tools you love" section to showcase plugin integrations:
- WordPress - Core CMS integration
- WooCommerce - E-commerce integration
- WPML - Multilingual support
- Gutenberg - Block editor compatibility
- Elementor - Page builder compatibility

## Technical Details

- All SVG icons are optimized for web display
- Icons use official brand colors where applicable
- Icons are sized consistently (h-6 w-6 classes)
- Grayscale hover effect for elegant presentation
- Scalable vector format for crisp display at any size
