# Favicon and SEO Assets

This directory contains favicon files and SEO assets for the GitHub Pages site.

## Files

- favicon.ico - Main favicon for browsers
- favicon-16x16.png - Small favicon
- favicon-32x32.png - Medium favicon
- apple-touch-icon.png - Apple touch icon
- manifest.json - Web app manifest

## Usage

These files are referenced in the HTML head sections of all pages for:
- Browser bookmarks and tabs
- Mobile app icons
- SEO optimization
- Web app installation
