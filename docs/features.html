Performance

Indexed Custom Tables

Skip the slow `wp_postmeta` table. We use custom SQL tables indexed specifically for relationships, enabling you to query millions of connections in milliseconds without caching plugins.

SELECT * FROM wp_ncr_relationships
WHERE parent_id = 1402
AND child_type = 'product';

code Developer First

Seamless WP_Query Integration

Stop writing complex raw SQL JOINs. Use our simplified meta_query-like syntax to fetch related content efficiently while maintaining standard WordPress development patterns. 

$args = [
  'post_type'  => 'book',
  'connected_to' => [
    'relationship' => 'author_to_book',
    'from'         => 123, // Author ID
  ],
  'posts_per_page' => 10,
];

// Standard WP_Query
$query = new WP_Query( $args );
