# Elementor

The plugin registers **Dynamic Tags** for Elementor so you can output related content in Elementor-built pages.

## Dynamic Tags

- **Related Posts** — Choose relation type, limit, order, and output format (IDs, titles, links, etc.).
- **Related Users** — e.g. users linked via favorites; optional avatar and display name.
- **Related Terms** — Related taxonomy terms with optional output format.

## Usage

1. Edit a template or page with Elementor.
2. Add a Dynamic Tag where you want related content (e.g. in a Text or HTML widget).
3. Select the NCR tag (Related Posts, Related Users, or Related Terms), pick the relation type and options.

Direction (incoming/outgoing) and layout options are available in the tag settings.
