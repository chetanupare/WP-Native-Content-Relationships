Developer Documentation

Complete guide to integrating Native Content Relationships into your WordPress projects. Learn how to create, manage, and query content relationships with our powerful API.

Installation

From WordPress.org

  1. Navigate to Plugins → Add New in your WordPress admin
  2. Search for "Native Content Relationships"
  3. Click Install Now and then Activate

From GitHub

  1. Download the plugin from the GitHub repository
  2. Upload the native-content-relationships folder to /wp-content/plugins/
  3. Activate the plugin in WordPress admin

Basic Usage

Creating Relationships

// Create a simple post-to-post relationship
$relation_id = wp_add_relation( 123, 456, 'related_to' );

// Create a user-to-post relationship
$relation_id = wp_add_relation( 15, 123, 'favorite_posts', null, 'post' );

// Create a post-to-term relationship
$relation_id = wp_add_relation( 123, 25, 'categorized_as', null, 'term' );

Querying Related Content

// Get related posts
$related_posts = wp_get_related( 123, 'related_to' );

// Get user's favorite posts
$favorite_posts = wp_get_related( 15, 'favorite_posts', array(), 'post' );

// Get posts categorized with a specific term
$categorized_posts = wp_get_related( 25, 'categorized_as', array(), 'term' );

Removing Relationships

// Remove a specific relationship
wp_remove_relation( 123, 456, 'related_to' );

// Remove all relationships of a type from a post
wp_remove_relation( 123, null, 'related_to' );

Relationship Types

Built-in Types

  • related_to - Bidirectional relationship between posts
  • parent_of - Unidirectional parent-child relationship
  • references - Unidirectional reference relationship

Custom Types

You can register custom relationship types:

function register_custom_relation_type() {
    naticore_register_relation_type( 'sponsored_by', array(
        'label' => 'Sponsored By',
        'bidirectional' => false,
        'allowed_post_types' => array( 'post', 'page' ),
        'supports_users' => true,
        'supports_terms' => false,
    ) );
}
add_action( 'init', 'register_custom_relation_type' );

API Reference

wp_add_relation()

Creates a new relationship between two content items.

wp_add_relation( $from_id, $to_id, $type, $direction, $to_type )

Parameters

  • $from_id (int) - Source content ID
  • $to_id (int) - Target content ID
  • $type (string) - Relationship type
  • $direction (string) - Direction: 'unidirectional' or 'bidirectional'
  • $to_type (string) - Target type: 'post', 'user', or 'term'

Return

int|WP_Error - Relationship ID on success, WP_Error on failure

wp_get_related()

Retrieves related content items.

wp_get_related( $post_id, $type, $args, $to_type )

Parameters

  • $post_id (int) - Source post ID
  • $type (string) - Relationship type
  • $args (array) - Query arguments
  • $to_type (string) - Target type: 'post', 'user', or 'term'

Return

array|WP_Error - Array of related items or WP_Error on failure

wp_remove_relation()

Removes a relationship between content items.

wp_remove_relation( $from_id, $to_id, $type, $to_type )

Parameters

  • $from_id (int) - Source content ID
  • $to_id (int) - Target content ID (null to remove all)
  • $type (string) - Relationship type
  • $to_type (string) - Target type: 'post', 'user', or 'term'

Return

bool|WP_Error - True on success, WP_Error on failure

Hooks & Filters

Actions

naticore_relation_added

Fired after a relationship is created.

do_action( 'naticore_relation_added', $relation_id, $from_id, $to_id, $type );

naticore_relation_removed

Fired after a relationship is removed.

do_action( 'naticore_relation_removed', $from_id, $to_id, $type );

Filters

naticore_content_relations_allowed

Filter whether relationship queries are allowed.

apply_filters( 'naticore_content_relations_allowed', $allowed, $post_id, $type );

naticore_get_related_args

Filter query arguments for getting related content.

apply_filters( 'naticore_get_related_args', $args, $post_id, $type );

REST API

Endpoints

GET /naticore/v1/post/{id}

Get relationships for a specific post.

curl -X GET "https://example.com/wp-json/naticore/v1/post/123"

POST /naticore/v1/relationships

Create a new relationship.

curl -X POST "https://example.com/wp-json/naticore/v1/relationships" \
  -H "Content-Type: application/json" \
  -d '{"from_id": 123, "to_id": 456, "type": "related_to"}'

DELETE /naticore/v1/relationships

Remove a relationship.

curl -X DELETE "https://example.com/wp-json/naticore/v1/relationships" \
  -H "Content-Type: application/json" \
  -d '{"from_id": 123, "to_id": 456, "type": "related_to"}'

WP-CLI

Commands

List Relationships

wp content-relations list --post=123 --type=related_to --format=json

Add Relationship

wp content-relations add 123 456 --type=related_to

Remove Relationship

wp content-relations remove 123 456 --type=related_to

Count Relationships

wp content-relations count --post=123

Integrations

WooCommerce

// Get related products
$related_products = wp_get_related_products( $product_id, 'related_product' );

// Add product to upsell collection
wp_add_relation( $product_id, $upsell_id, 'upsell' );
