Developer Documentation

Welcome to the developer hub. Learn how to extend Native Content Relationships programmatically using our robust API and hooks system.

The Native Content Relationships plugin allows you to define strict or loose relationships between any WordPress post types. While the UI offers extensive controls, the real power lies in the code-first approach for themes and complex integrations.

info
Note: Ensure you have version 2.4.0 or higher installed to use the functions described below.

Quick Start

Recommended

To register a relationship programmatically, use the ncr_register_relationship function hooked into WordPress's init action.

functions.php 
add_action('init', function() {
    // Register a Many-to-Many relationship between Books and Authors
    ncr_register_relationship('books', 'authors', [
        'cardinality' => 'many-to-many',
        'reciprocal'  => true,
        'meta'        => [
            'role' => 'string'
        ]
    ]);
});
